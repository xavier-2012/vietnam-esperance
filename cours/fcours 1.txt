1?? Créer une partie rectangulaire (section)
Commande HTML
<section class="section"></section>
Explication
    • <section> = une grande partie du site
    • C’est sémantiquement mieux qu’un <div> pour la structure
    • Chaque section = un rectangle

2?? Mettre du contenu dans la section
<section class="section">
  <h2>Titre de la partie</h2>
  <p>Texte de la partie</p>
</section>
Explication
    • h2 ? titre principal de la section
    • p ? contenu
    • Tout reste dans un même rectangle

3?? Donner une largeur propre et centrée
CSS
.section {
  max-width: 1200px;
  margin: 40px auto;
}
Explication
    • max-width ? largeur du rectangle
    • auto ? centre horizontalement
    • 40px ? espace entre les sections

4?? Créer la forme rectangulaire visible
.section {
  background-color: #ffffff;
  padding: 40px;
}
Explication
    • background-color ? fond blanc
    • padding ? espace intérieur
    • Le rectangle devient visible

5?? Ajouter une bordure (optionnel)
border: 1px solid #ddd;
Explication
    • Trace le contour du rectangle
    • Utile pour structurer visuellement

6?? Arrondir les angles (moderne)
border-radius: 12px;
Explication
    • Coins arrondis
    • Donne un rendu propre et actuel

7?? Ajouter une ombre pour détacher du fond
box-shadow: 0 6px 20px rgba(0,0,0,0.08);
Explication
    • Effet “carte”
    • Sépare clairement chaque partie

8?? Couleur de fond du site (important)
body {
  background-color: #f0f2f5;
}
Explication
    • Le fond gris clair fait ressortir les rectangles blancs
    • Très utilisé sur les sites pros

9?? Créer plusieurs parties rectangulaires
<section class="section">Accueil</section>
<section class="section">Notre histoire</section>
<section class="section">Nos actions</section>
<section class="section">Contact</section>
Explication
    • Chaque <section> = un grand rectangle
    • Organisation claire du site

? Séparer visuellement les titres
.section h2 {
  border-bottom: 3px solid green;
  padding-bottom: 10px;
}
Explication
    • Ligne sous le titre
    • Structure nette et lisible

? Logique globale
    • body ? fond général
    • <section> ? gros rectangles
    • max-width + margin ? centrage
    • padding + shadow ? confort visuel
